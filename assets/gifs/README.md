# GIF Assets Guide

This folder stores product demo GIFs used in the root README.

## Required filenames

- `cli-diff.gif`
- `traffic-capture.gif`
- `replay-report.gif`
- `dashboard-overview.gif`

## Recording checklist

- Keep each GIF under 8-12 seconds.
- Use 1280px width or lower for reasonable size.
- Prefer one action per GIF (clear before/after).
- Redact any secrets, tokens, or private URLs.

## Suggested content

- `cli-diff.gif`: run `surge diff --help` and a sample diff command.
- `traffic-capture.gif`: show proxy ingest + stored traffic list.
- `replay-report.gif`: launch replay session + display drift summary.
- `dashboard-overview.gif`: show dashboard navigation and key metrics.
